This is a learning plan for studying FPGA.

You can also jump right in by:
- Running `nix-shell` to set up the environment (what's nix? I'm glad you asked.)
- Delete the individual project folders, or just the `top.v` files. The folder contains a `NOTES.md` that tracks what I learnt.
- Tell it to read the lesson plan and start teaching you!

It should set up lots of the files you need, but not write the `top.v` file. That's your job to learn.

Try to do the work yourself, but if you get stuck, ask it to explain, give you hints.
